To compile the executable binary, please type:
    1. `make clean`
    2. `make`
The executable binary `fm` will store under directory `./bin`

To run the partition on circuit, please type:
    1. `./bin/fm <input_file_path> <output_file_path>`
For example, to run program on input_1, please type:
    1. `./bin/fm input/input_1.dat output/output_1.dat`
